
## 📱 Overview
SmartSpend is a iOS mobile finance application developed for my Final Year Project to help usee manage their finances better. It has spending analysis on dashboard (home page) where users can see insights on their spendings

## 🎯 Objectives
To offer insights by creating visual reports – this app is going to show the information of spendings by using pie charts and bar charts. This will help users to see where they are spending their money. 

To help users save their money more responsibly by keeping tracking of expenses – users can track on how much money they are spending in different sections and make better financial decisions, hence why the apps offer wisely spendings. 

To decrease the time a user takes to track their spendings  - using spreadsheets or logbooks to track budgets can be very tedious. This app will categorize expenses smoothly. 

To raise their consistent saving habits compared to other traditional budgeting like Excel Spreadsheets – unlike manual spreadsheets, the app offers an intuitive goal-setting feature and progressive tracking 

To manage their budget more wisely through a user-friendly user interface – this user interface makes sure that users can navigate through the services without them being overwhelmed.  

## 🛠️ Technologies Used
- Swift (the langugage)
- SwiftUI (
- Xcode (the IDE software)
- Figma (a software for making prototype)

## ✨ Features
- Add, edit, and delete expenses (Expenses Page)
- Categorize spending (Budgeting Page)
- View monthly summary (Dashboard Page)
- Track savings progress (Saving Goals Page)
